# Parrot-in-Termux2
A new version of parrot in termux added new features such as extra arrows and shortcut keys and new font color.

## Installation
1. `pkg install git -y`
2. `git clone https://github.com/Mrp1r4t3/Parrot-in-Termux2`
3. `cd Parrot-in-Termux2`
4. `sh install.sh`
5. `sh parrot.sh`
